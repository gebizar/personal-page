Projects

Entree.ai

My primary task on my Capstone Senior project was to implement a database for use with Google Home's so people could order intuitively, more sentence-like, than a typical option tree. I facilitated the creation of an API that double checked to see if menu items were in a restaurants database prior to confirming the order. In short, I created the backend to ensure that after the language was parsed the item even existed. It was intensely challenging and in conjunction with other developers using language parsing libraries such as spaCy, it really brought to light just how insanely difficult it is to work with language parsing.
